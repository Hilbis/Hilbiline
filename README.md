# Hilbiline
> A readline-inspired line editor made for Hilbish

Hilbiline is the custom made line editor for the
[Hilbish Shell](https://github.com/Hilbis/Hilbish).
It is a fork of [go-cli](https://github.com/deadsy/go-cli)
